Buyer's Documentation Manual

Welcome to the buyer's guide. This page outlines the simple, secure process of purchasing items through our Discord integration and web checkout system.

The Purchase Workflow

The buying process is designed to be seamless, taking you directly from a Discord notification to a secure payment page.

Step-by-Step Guide

  1. Locate an Item: Find an item for sale in the designated Discord channel.
  2. Click "Buy Now": Click the link provided in the bot's message embed.
  3. Secure Checkout: You will be redirected to a secure c500-web-go checkout page in your browser. Review order details.
  4. Complete Payment: Click "Pay with Stripe" to finalize the transaction securely.
  5. Confirmation: You will receive an on-screen confirmation and a receipt via email.
